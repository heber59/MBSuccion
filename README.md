# MBAireYSuccion

Sitio base en Astro para `MBAireYSuccion`, orientado desde el inicio a SEO tecnico, claridad semantica y futura expansion para posicionamiento organico.

## Objetivo

Crear una presencia web que:

- explique con claridad la marca y sus servicios;
- facilite el rastreo e indexacion en buscadores;
- mejore la comprension por parte de herramientas LLM;
- sirva como base para crecer con paginas de servicios, ciudades y contenido editorial.

## Base SEO incluida

- layout con `title`, `description`, `canonical` y etiquetas Open Graph;
- datos estructurados `Organization`, `WebSite` y `FAQPage`;
- homepage con jerarquia semantica real;
- archivos `robots.txt` y `llms.txt`;
- estructura lista para escalar con SEO local y contenido por intencion de busqueda.

## Pendientes recomendados

- definir dominio final y actualizar `astro.config.mjs`;
- crear paginas por servicio;
- crear paginas por ciudad objetivo;
- añadir testimonios, casos de exito y datos de contacto reales;
- generar una estrategia de contenidos basada en keywords reales.

## Comandos

```sh
npm install
npm run dev
npm run build
```
